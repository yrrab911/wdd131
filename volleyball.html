Site Plan for BYU Volleyball Group Finder

Site Name

BYU Volleyball Group Finder

This name clearly indicates the purpose of the site: to help BYU students find and join volleyball groups. It is relevant to the target audience and easy to remember.

Site Purpose

The site serves as a hub for BYU students to connect with others interested in playing volleyball. It will provide information about volleyball rules, allow users to create and join games, and foster a community among players.

Scenarios

1. What days and times are there volleyball games scheduled this week?

2. How can I create a new game post to invite others to join?

Color Schema

Primary Color: #007BFF (Used for headings and buttons)

Secondary Color: #28A745 (Used for accents and highlights)

Typography

Headings: Montserrat, sans-serif

Body: Arial, sans-serif
